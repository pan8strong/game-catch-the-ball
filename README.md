Cara Menjalankan Proyek
1. Clone repo ini:
git clone https://github.com/username/flutter-project.git
2. Buka di Android Studio
3. Jalankan flutter pub get
4. Pilih emulator atau device kamu
5. Klik tombol ▶️ Run
